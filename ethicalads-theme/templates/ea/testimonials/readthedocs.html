About Read the Docs

Read the Docs needed a solution to monetize a large user base but wasn't willing to compromise on privacy.

Read the Docs had a huge audience but suffered a common problem open source projects usually have which is a lack of revenue and support. A service that millions depended on relied on its developers wearing pagers at their day jobs.

EthicalAds was built specifically for Read the Docs (we are the same company after all). Their users knew that advertising would bring a better service but neither Read the Docs nor its users wanted third-party tracking from a large ad network.

The revenue from advertising through EthicalAds allowed Read the Docs to have a small team of developers work on it full-time without taking venture capital.

Read the Docs hosts technical documentation for open source projects and provides a continuous integration solution to build documentation from source control on every commit.
